API Reference

Pybtex.BibEntry — Type

This object represents a BibTeX entry. It is a wrapper around pybtex's pybtex.database.Entry object, connecting it with its corresponding key.

source
Pybtex.BibLibrary — Type

This provides a simple interface to read and manipulate bibtex databases.

source
Pybtex.BibType — Type

This module provides the types used in the BibLibrary module. These are the general BibTeX types useful for dispatching entries on them.

source

Pybtex.getAuthors — Method

This function returns the authors of a BibTeX entry. The authors are returned as a vector of PersonName objects.

If the authors field is not present, but editors are found, the editors are returned instead.

Pybtex.journalAbbreviationFromTag — Method

Returns the journal abbreviation from the tag. For example, the tag "apj" returns "Astrophys. J.".

source
Pybtex.journalFullNameFromTag — Method

Returns the journal full name from the tag. For example, the tag "apj" returns "Astrophysical Journal".

source
Pybtex.journalTagFromFullName — Method

Returns the journal tag from the full name. For example, the full name "Astrophysical Journal" returns "apj". This is essentially an inversion of journalFullNameFromTag.
